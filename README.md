# JuliaLab

## Structure

- `papers/` - Research papers and literature
- `bib/` - Bibliography files
- `src/` - Source code
- `data/` - Raw and processed datasets
- `output/` - Generated figures and results
- `spec/` - Spec Kit artifacts (constitution, spec, plan, tasks)
- `notes/` - Portable notes (if enabled)

## Getting Started

1. Review the project constitution in `spec/constitution.md` (if created)
2. Use eurAIka commands from the command palette (Ctrl+Shift+P)
3. Start a Socratic session to refine your research question

## eurAIka Commands

- **Create Whiteboard Note**: Document ideas and concepts
- **Search Literature**: Find relevant papers
- **Start Socratic Session**: Clarify research questions
- **Show Code Map**: Visualize knowledge graph
